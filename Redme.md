# JavaScript Practice Guide for Vue.js & Next.js

This repository contains **practice exercises** to strengthen your **modern JavaScript (ES6+)** skills for Vue.js and Next.js development.

---

## Instructions

1. **Clone this repository** to your local machine:
```bash
git clone <repo-url>
cd <repo-folder>
````

2. **Create a JS file** for each exercise category:

```bash
touch variables.js loops.js functions.js arrays.js objects.js dom.js
```

3. **Follow the exercises below**, write your code, and run it using Node.js:

```bash
node variables.js
```

---

## Exercises

### 1. Variables & Data Types

* Declare `let`, `const`, and `var` variables and log them.
* Create an object `user` with properties `name`, `age`, `email`.
* Create an array of your 5 favorite items.
* Practice type conversions: number → string, string → number, boolean → number.

### 2. Operators & Conditions

* Compare numbers using `>`, `<`, `===`, `!==`.
* Write an `if-else` to check if a number is positive, negative, or zero.
* Use a ternary operator to print "Even" or "Odd".
* Write a switch case for weekdays (1-7).

### 3. Loops

* Print numbers 1 to 10 using a `for` loop.
* Print numbers 1 to 10 using a `while` loop.
* Iterate over an array using `for-of`.
* Iterate over an object using `for-in`.

### 4. Functions

* Write a function to add two numbers.
* Write an arrow function to square a number.
* Write a function with default parameters.
* Write a function that takes rest parameters and returns the sum.

### 5. Arrays

* Create an array of 5 numbers.
* Use `map` to double each number.
* Use `filter` to keep only even numbers.
* Use `reduce` to sum all numbers.
* Add an item to the end and remove the first item.

### 6. Objects

* Create an object `book` with properties `title`, `author`, `pages`.
* Access properties using dot notation and bracket notation.
* Destructure object properties into variables.
* Add a method `describe()` that returns a string with book details.

### 7. ES6+ Features

* Use template literals to print dynamic messages.
* Use spread operator to combine arrays.
* Use rest operator in functions.
* Practice import/export modules (create two files).

### 8. DOM & Events (Optional, for browser practice)

* Select an element using `document.querySelector`.
* Change its text content.
* Add a click event listener to log a message.
* Create a list of buttons dynamically from an array.

---

## How to Practice

1. Complete **one exercise at a time** in a separate `.js` file.
2. Run it in Node.js (or in the browser for DOM exercises).
3. Check the output in console.
4. Refactor using **modern JS features** (`let`, `const`, arrow functions, destructuring).
5. Repeat exercises until comfortable.

---

## Tips

* Use **console.log** to debug and understand outputs.
* Try **combining exercises** to make small projects (e.g., Todo list, Calculator).
* Focus on **writing clean and modern JS code**, as this will help in Vue.js and Next.js projects.
